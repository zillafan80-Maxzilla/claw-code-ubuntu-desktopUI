# PreTeXt

This is a local reference skill built from the official GitHub repository:

- `references/pretext`

## What It Covers

Use this skill when the task involves:

- PreTeXt books, articles, monographs, and publishing workflows
- XML-like source authoring and multi-format output generation
- XSL transform behavior, schema validation, and publication pipelines
- official example projects and CLI/helper tooling
