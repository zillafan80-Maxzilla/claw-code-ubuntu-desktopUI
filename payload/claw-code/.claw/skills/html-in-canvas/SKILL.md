# HTML-in-Canvas

This is a local reference skill built from the official GitHub repository:

## What It Covers

Use this skill when the task involves:

- the WICG HTML-in-Canvas proposal
- rendering HTML content inside a canvas-like surface
- OffscreenCanvas and rendering architecture questions
- interaction, accessibility, and event model tradeoffs
- implementation notes, explainer content, and spec discussion
